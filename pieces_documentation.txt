# Azure OpenAI

Powerful AI tools from Microsoft

Minimum Supported Release: 0.36.1
Authors: MoShizzle, abuaboud

---

## Flow Structure Example

### Flow Metadata
```json
{
  "name": "Vendor Data Validation Agent",
  "description": "",
  "tags": [
  ],
  "pieces": [
    "@activepieces/piece-webhook",
    "@activepieces/piece-inputs",
    "@activepieces/piece-google-drive",
    "@activepieces/piece-zbrain-ai",
    "@activepieces/piece-http",
    "@activepieces/piece-store",
    "@activepieces/piece-utilities",
  ],
  "template": {
    "displayName": "Vendor Data Validation Agent"
  }
}
```

### Flow Overview
This flow demonstrates the following sequence:
1. Starts with a webhook trigger
2. Executes @activepieces/piece-inputs piece with action 'long_text'
3. Executes @activepieces/piece-store piece with action 'remove_value'
4. Executes @activepieces/piece-utilities piece with action 'tokenizer'
6. Executes @activepieces/piece-store piece with action 'remove_value'

### Complete Flow Configuration
```json
{
  "trigger": {
    "type": "WEBHOOK",
    "settings": {},
    "nextAction": {
        "name": "step_1",
        "type": "PIECE",
        "settings": {
          "pieceName": "@activepieces/piece-inputs",
          "actionName": "long_text",
          "packageType": "ARCHIVE",
          "pieceType": "CUSTOM",
          "errorHandlingOptions": {
            "retryOnFailure": {"value": false},
            "continueOnFailure": {"value": false},
          },
          "input": {
            "input": "{{trigger.body.file.content}}",
          }
        }
        ,"nextAction": {
            "name": "step_3",
            "type": "PIECE",
            "settings": {
              "pieceName": "@activepieces/piece-store",
              "actionName": "remove_value",
              "packageType": "REGISTRY",
              "pieceType": "OFFICIAL",
              "errorHandlingOptions": {
                "retryOnFailure": {"value": false},
                "continueOnFailure": {"value": false},
              },
            }
            ,"nextAction": {
                "name": "step_6",
                "type": "PIECE",
                "settings": {
                  "pieceName": "@activepieces/piece-utilities",
                  "actionName": "tokenizer",
                  "packageType": "ARCHIVE",
                  "pieceType": "CUSTOM",
                  "errorHandlingOptions": {
                    "retryOnFailure": {"value": false},
                    "continueOnFailure": {"value": false},
                  },
                  "input": {
                    "text": "{{step_1}}",
                  }
                }
                ,"nextAction": {
                    "name": "step_14",
                    "type": "PIECE",
                    "settings": {
                      "pieceName": "@activepieces/piece-store",
                      "actionName": "remove_value",
                      "packageType": "REGISTRY",
                      "pieceType": "OFFICIAL",
                      "errorHandlingOptions": {
                        "retryOnFailure": {"value": false},
                        "continueOnFailure": {"value": false},
                      },
                    }
                    }
    }
    }
    }
    }
  }
}
```

### Variable References
The following variables are passed between steps:

**Step `step_1`**
- Uses `{{trigger.body.file.content}}` for input `input`

**Step `step_6`**
- Uses `{{step_1}}` for input `text`
